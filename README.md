# Giftyy

Giftyy is a mobile application that allows users to record and attach a video message to a gift.

## Features

*   Record a short video message.
*   Attach the video to a gift.
*   Playback the video message.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js
*   npm
*   Expo CLI

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/giftyy.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the app
    ```sh
    npx expo start
    ```
